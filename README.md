# Portifolio_Backyardigans
Este repositório é destinado a atividade de **DevOps**.
## Objetivo do Projeto:
Criar um site portfólio coletivo, reunindo as páginas pessoais de todos os integrantes do grupo em uma única aplicação web.

Cada integrante ficou responsável por:

Criar sua página pessoal.

Integrar seus arquivos ao projeto.

Atualizar o menu principal com o link para seu portfólio.

Seguir o fluxo correto de versionamento (pull → commit → push).

## Tecnologias Utilizadas:
    HTML5
    CSS3
    Git
    GitHub
    GitHub Pages

## Como executar o projeto localmente:

1- 
```bash
git clone https://github.com/JoaoSouzera/Portifilio_Backyardigans.git
``` 
2 - Acessar a pasta do projeto.

3 - abrir o **index.html** no navegador.
